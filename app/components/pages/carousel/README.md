### scroll-smooth(scroll-behavior: smooth)

- プログラムによるスクロールを滑らかにアニメーション化
- JavaScriptでのスクロール操作に影響
- つまり、左右ボタンクリック時の移動を瞬間移動ではなく滑らかのアニメーションで移動させることができる。
